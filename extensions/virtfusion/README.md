# VirtFusion

This is a Raycast extension for [VirtFusion](https://virtfusion.com/) instances.

## ğŸš€ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/virtfusion) OR `install` via Raycast Store

2. **Enter your Panel Details**:

    The extension allows you to manage multiple panels at the same time. For each panel, do the following:

    a. `Run` "Manage Panels" then "Add Panel"

    b. `Enter` full URL e.g. https://virtfusion.example.com
    
    c. `Enter` "API Key" by generating it from "Account" > "API" > "Generate Token"
