# Transport NSW Changelog

## [Initial Release] - 2026-02-16

- Plan trips between stations with multiple journey options
- View detailed stop-by-stop journey information with platforms and times
- Save favorite trips for quick access
- Menu bar command showing upcoming departures for saved trips
- AI-powered natural language trip planning and delay checking
- Real-time delay information from Transport NSW APIs
- Support for Sydney Trains, Metro, and Light Rail
