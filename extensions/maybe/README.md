# Maybe

This is a [Raycast](https://raycast.com) extension for [Maybe](https://github.com/maybe-finance/maybe) - _The personal finance app for everyone_.

With this extension you can:

- Search Accounts
- Search Transactions
    - Create
    - Delete

## ðŸš€ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/maybe) OR `install` via Raycast Store

2. **Configure extension**: The first time you use the extension, you'll need to enter "URL" and "API Key" in Preferences OR at first prompt:

    - **Maybe URL** - URL of Maybe instance (e.g. https://maybe.example.com/)
    - **Maybe API Key** - API Key from Settings > API Keys
