# Proton Pass Client Changelog

- Fixed error when accessing BrowserExtension on unsupported environments

## [Fix & Improvements] - 2026-02-19

- Search and filter items by status (Active/Trashed) and vault
- Copy item fields quickly with shortcut actions
- Background refresh and caching for faster results
- Support for web extension integration
